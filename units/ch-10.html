Ünite 10: Mantıksal Programlama (Prolog)

1) Mantıksal programlama nedir?

Fikir: “Nasıl yaparım?” yerine “Ne doğrudur?” ve “Bu doğrulardan hangi sonuçlar çıkar?” sorulur.

Program, iki ana parçadan oluşur:

  • Fact (Olgu): Doğru kabul edilen temel bilgi.
  • Rule (Kural): Yeni bilgi çıkarımı yapan çıkarım şablonu.

Çözüm, Prolog'da genellikle unification (birleştirme) ve backtracking (geri izleme) ile bulunur.

2) Fact + Rule örneği (çıkmış soruyla uyumlu)

% Fact (olgu)
insan(selim).

% Rule (kural): X insan ise X ölümlüdür
olumlu(X) :- insan(X).

% Query (sorgu)
?- olumlu(selim).
% true

Bu ne yaptı? Sorgu olumlu(selim) geldi. Kuralın sol tarafı ile eşleştirdi, X = selim atadı (unification). Sonra sağ tarafa indi: insan(selim) fact olarak veritabanında bulundu → başarı.

3) Backtracking mantığı (neden “birden fazla cevap” çıkabilir?)

Prolog aynı sorgu için birden çok çözüm bulabilir. Bunun nedeni kural/fact veritabanında birden çok eşleşme olabilmesidir.

insan(ali).
insan(ayse).

?- insan(X).
% X = ali ;
% X = ayse ;
% false.

; (noktalı virgül) bir sonraki çözümü istediğin anlamına gelir. Prolog geri izleyerek (backtracking) başka eşleşme arar.

4) Python ile benzetim (mantığı anlamak için)

Sınav Prolog isteyebilir; ama mantığı kavramak için Python'da çok basit bir “fact listesi + rule” taklidi yapılabilir.

# Fact'ler
insan = {"selim", "ali", "ayse"}

def olumlu(x):
    # Rule: olumlu(X) :- insan(X).
    return x in insan

print(olumlu("selim"))  # True
print(olumlu("veli"))   # False

Neden yazdım? “rule = koşul sağlanıyorsa sonuç doğru” mantığını hızlı oturtmak için. (Soru Prolog isterse Prolog kodu asıl cevaptır.)

Sık yapılan hata

  • Prolog'u “if‑else” gibi düşünmek. Prolog'da temel araç eşleştirme + geri izlemedir.
  • Değişkenleri (X, Y) “değiştirilebilir bellek hücresi” sanmak. Prolog değişkeni çoğu senaryoda bağlanır ve geri izleme ile farklı bağlanır.

Mini test

  1. Fact ile Rule arasındaki farkı 1 cümleyle yaz.
  2. Unification nedir? 1 örnekle anlat.
  3. Backtracking ne zaman devreye girer?
