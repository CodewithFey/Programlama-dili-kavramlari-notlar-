Ünite 6 — Prosedürler, Fonksiyonlar ve Metotlar (Procedures, functions and methods)

Bu ünite, sınavda “metot çağırma”, “parametre geçirme”, “stack/heap”, “recursion” gibi soruların temelidir.

6.1 Introduction — Giriş

  • Procedure: genelde değer döndürmeyen alt program (C’de void fonksiyonlar).
  • Function: değer döndüren alt program.
  • Method: bir nesnenin/sınıfın içinde tanımlı alt program (OOP).

6.2 Parameters — Parametreler

Parametre geçirme yöntemlerini “değer mi gidiyor, adres mi gidiyor?” diye düşün.

C: Call by value (varsayılan)

#include <stdio.h>

void degistir(int x){ // x kopya
    x = 99;
}

int main(){
    int a = 10;
    degistir(a);
    printf("%d\n", a); // 10 (değişmez)
}

C: Pointer ile “referans etkisi”

#include <stdio.h>

void degistir(int *p){ // p adres taşır
    *p = 99;           // asıl değişkeni değiştirir
}

int main(){
    int a = 10;
    degistir(&a);
    printf("%d\n", a); // 99
}

C++: Call by reference

#include <iostream>
using namespace std;

void degistir(int &x){
    x = 99;
}

int main(){
    int a = 10;
    degistir(a);
    cout << a; // 99
}

Python: isim bağlama + mutable/immutable farkı

def f(x):
    x = 99      # x sadece yerel isim; dışarıdaki sayıyı değiştirmez

a = 10
f(a)
print(a)        # 10

def g(lst):
    lst[0] = 99 # liste mutable -> dışarıdaki obje değişir

b = [10, 20]
g(b)
print(b)        # [99, 20]
Sınav cümlesi: C’de parametreler değerle iletilir; “orijinali değiştirmek” için pointer kullanırsın. C++’ta değer + referans iletimi doğrudan dil desteğidir.

6.3 Functions — Fonksiyonlar

  • Saf fonksiyon: yan etkisiz (test etmesi kolay).
  • Yan etkili fonksiyon: global değişken, IO, pointer ile durum değiştirir.

6.4 Storage management — Saklama yönetimi (Stack vs Heap)

  • Stack: fonksiyon çağrıları için otomatik alan (yerel değişkenler).
  • Heap: dinamik bellek (malloc/new), sen yönetirsin.
#include <stdlib.h>

int main(){
    int x = 5;               // stack
    int *p = malloc(sizeof(int)); // heap
    *p = 42;
    free(p);
}

6.5 Recursion — Özyineleme

Fonksiyon kendi kendini çağırır. 2 şart:

  • Base case (durma koşulu)
  • Her çağrıda base case’e yaklaşma
int fact(int n){
    if(n <= 1) return 1;
    return n * fact(n-1);
}

6.6 Forward references — İleriye referans

C’de fonksiyonun daha sonra tanımlansa bile önce “prototip” gerekir.

int f(int x); // prototip

int main(){ return f(3); }

int f(int x){ return x * 2; }

6.7 Subprograms as parameters — Alt programı parametre yapmak

C’de bu, function pointer ile yapılır. Python’da ise fonksiyonlar zaten birinci sınıftır.

#include <stdio.h>

int kare(int x){ return x*x; }

int uygula(int (*fp)(int), int v){
    return fp(v);
}

int main(){
    printf("%d\n", uygula(kare, 5)); // 25
}
def kare(x): return x*x
def uygula(fp, v): return fp(v)
print(uygula(kare, 5))
