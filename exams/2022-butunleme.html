03/02/2022 — Bütünleme (Çıkmış Sorular)

Bu bütünlemede C ağırlıklı temel kavramlar + OOP kavramları var. Her soruda önce “soru ne ister” netleştirilir.

Soru 1 — C'de Dikdörtgen struct tanımı ve değer atama

İstenen: yukseklik ve genislik alanlarına sahip bir veri tipi tanımla (struct) ve bir değişkene değer ata.

#include <stdio.h>

typedef struct {
    float yukseklik;
    float genislik;
} Dikdortgen;

int main() {
    Dikdortgen d1;

    // alanlara değer atama
    d1.yukseklik = 12.5f;
    d1.genislik  = 25.0f;

    printf("y=%.2f g=%.2f\n", d1.yukseklik, d1.genislik);
    return 0;
}

Python karşılığı (basit “struct benzeri”):

from dataclasses import dataclass

@dataclass
class Dikdortgen:
    yukseklik: float
    genislik: float

d1 = Dikdortgen(12.5, 25.0)
print(d1)

Soru 2 — C ve C++ parametre geçirme yöntemleri

İstenen: C ve C++'ta parametre geçirme biçimlerini örnekle anlat.

  • C: Call by value; ama adres geçirerek (pointer) “call by reference benzeri” etki elde edilir.
  • C++: Call by value + call by reference (ref) + pointer da var.
#include <stdio.h>

// 1) Değer iletimi: a kopya
void swap_value(int a, int b) {
    int tmp = a; a = b; b = tmp;
}

// 2) Adres iletimi: asıl değişkenler değişir
void swap_ptr(int *a, int *b) {
    int tmp = *a; *a = *b; *b = tmp;
}

int main() {
    int x = 3, y = 7;
    swap_value(x, y);
    printf("value: x=%d y=%d\n", x, y); // değişmez

    swap_ptr(&x, &y);
    printf("ptr:   x=%d y=%d\n", x, y); // yer değişir
}
#include <iostream>
using namespace std;

void swap_ref(int &a, int &b) { // referans iletimi
    int tmp = a; a = b; b = tmp;
}

int main() {
    int x = 3, y = 7;
    swap_ref(x, y);
    cout << "ref: x=" << x << " y=" << y << "\n";
}

Soru 3 — Bağımsız / ayrı / bağımlı derleme ve C hangisi?

İstenen: 3 terimi tanımla ve C’nin hangisine daha uygun olduğunu yaz.

  • Bağımsız derleme: Her modül tek başına derlenip test edilebilir; modüller arası tip/arayüz denetimi sınırlı olabilir.
  • Ayrı derleme (separate compilation): Kaynak dosyalar (.c) ayrı ayrı derlenir (.o) ve linker ile birleşir. C tipik örnektir.
  • Bağımlı derleme: Modüller sıkı bağımlıdır; küçük değişiklikte her şeyin yeniden derlenmesi gerekir (pratikte istenmez).

C için klasik yapı:

// fonksiyon.h  (arayüz)
#ifndef FONKSIYON_H
#define FONKSIYON_H
float alan(float g, float y);
float cevre(float g, float y);
#endif
// fonksiyon.c (uygulama)
#include "fonksiyon.h"
float alan(float g, float y) { return g * y; }
float cevre(float g, float y){ return 2 * (g + y); }
// main.c
#include <stdio.h>
#include "fonksiyon.h"
int main() {
    printf("%.2f\n", alan(5,3));
    printf("%.2f\n", cevre(5,3));
    return 0;
}

Derleme (gcc):

gcc -c fonksiyon.c     # fonksiyon.o
gcc -c main.c          # main.o
gcc main.o fonksiyon.o -o dikdortgen
./dikdortgen

Soru 4 — Value / reference tipi test etme (örnek kodla)

İstenen: Bir dilde bir değişkenin “değer tipi mi referans tipi mi” olduğunu davranıştan anla ve farkları yaz.

En pratik test: kopyala → değiştir → ilk değişken değişti mi?

C++ referans örneği

#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int &b = a;   // b, a'nın referansı
    b = 20;
    cout << a << "\n"; // 20
}

Python list örneği

a = [10, 20, 30]
b = a          # aynı listeyi işaret eder
b[0] = 1
print(a)       # [1, 20, 30]

Özet fark: Değer tipinde kopya taşınır; referans tipinde “aynı nesneyi gösteren adres/referans” taşınır.

Soru 5 — Overload ve Override

İstenen: Overload ve override’ı kodla ayır.

  • Overload (aşırı yükleme): Aynı isimli fonksiyon, farklı parametre listeleri. (compile-time)
  • Override (ezme): Alt sınıfın, üst sınıftaki virtual fonksiyonu yeniden tanımlaması. (run-time / dynamic binding)
#include <iostream>
using namespace std;

class Hesapla {
public:
    int topla(int a, int b) { return a + b; }           // overload 1
    double topla(double a, double b) { return a + b; }  // overload 2

    virtual void yazdir() { cout << "Hesapla\n"; }      // override edilebilir
};

class Hesaplayici : public Hesapla {
public:
    void yazdir() override { cout << "Hesaplayici\n"; } // override
};

int main() {
    Hesapla h;
    cout << h.topla(2,3) << "\n";
    cout << h.topla(2.5, 3.5) << "\n";

    Hesapla *p = new Hesaplayici();
    p->yazdir(); // "Hesaplayici" (dynamic binding)
    delete p;
}

Python notu: Python'da overload “imza” üzerinden otomatik yapılmaz (genelde default arg / *args ile çözümlenir). Override ise sınıf mirasında doğrudan yapılır.
